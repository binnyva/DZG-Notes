When someone is sharing, you can use this framework to help them share more/ actively listen.

- Acknowledge: Summarize what they said
- Compliment
- Ask: You can ask for more details about some point.

This helps implement the Listening -> Empathy -> Curiosity -> Listening -> ... Cycle.
